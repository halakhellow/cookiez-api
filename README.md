
# <p align="center">COOKIEZ API</p> 

### About the project:
An API for cookiez website which is a platform to connect customers who are looking for homemade food with chefs who offer that.

### Technologies used:
- NodeJS
- ExpressJS
- MongoDB
- Nodemailer
- Multer
- Firebase
